Storing CSS in External File

Basically most of the time we store CSS code in external file and link it in the HTML header section.
To use a external css file in the HTML document create a .css file and save it. Then use the code;

<link rel="stylesheet" type="text/css" href="demo.css" />
in head section of HTML document.

Network Tab

Network Tab is used to monitor the Network requests and to check how does your page load if you have slower or faster connection. You can open Network Tab by right clicking on any blank space on webpage and then clicking on "Inspect" option from menu and select "Network" Tab.

ID Selector

We can select a single HTML element and change its properties with id tag. To use id tag we have to give a unique name. Then we can declare it in any of the HTML element. For example: 

<p id="demo-id">
In the above example we have given id named "demo-id" to p element with id tag.
We can then change properites of ID using hastag "#" followed by the id name. 

      #demo-id {
      color: #02fff2;
      }

Working With Font-Size and Pixel:

We can set the different weight of the font and size of the font with font-weight and font-size property.

Using Other Fonts from Google Fonts:

We can import and use other imported fonts from the internet. We can simply search for the fonts in "Google Fonts", select the font and then copy the fonts link provided by the Google fonts in the right bar and paste it in the head section of HTML document. 

<link rel="stylesheet" type="text/css" href="//fonts.googleapis.com/css?family=Open+Sans"/>
We have imported and used Open-sans in the above example.

Displaying Code in Webpage:

Sometimes we need to display code in Webpage for this we use:
a. <pre> tag: to output code as a plain text not as HTML code. Whitespace is also prevented and not ignored.
b. To simply add "<" or ">" code we use:
i) & + lt + ; (Less Than)
ii) & + gt + ; (Greater than)

Note: Remove "+" sign and whitespace.
