More about HTML Layout and Positioning

HTML Tags:

  • <nav>

    • Previously we have learnt about some of the tags like <header>, <main> and <footer>. Today we learnt about new tag <nav> tag which is a container that contains navigation links to the next page.

CSS Properties:

  • text-transform

    • "text-transform" property helps to change the letters of HTML elements to either all Uppercase or all Lowercase.
    text-transform: Uppercase;

  • text-shadow

    • We have previously discussed about box-shadow property. "text-shadow" is similar to that box-shadow property but as the name indicates that box-shadow property was applied for box, whereas text-shadow property can be applied on text. Same as box-shadow it has four values the first one is offset along x-axis, second one is offset along y-axis, third one is blur amount and the fourth one is the color.
    text-shadow: 1px 1px 1px red;

CSS Flexbox

CSS Flexbox is introduced to position html elements more easily and accurately. It is basically a container that helps to position elements. An area of document laid out using flexbox is called flex container. To create a flex container, we set the value of area's container display property to flex or inline-flex. As soon as we do this the direct children of that container becomes flex items. After doing this now we can use different flex Properties.
display:flex;

  • flex-direction

    • The flex direction property defines in which direction the flex items are needed to be placed. This property can have one of the four different values either row or column or column-inverse or row-inverse.
    flex-direction: row;

  • flex-wrap

    • It defines whether the flex items are forced in a single line or can be flowed into multiple lines. The flex-wrap property accepts two different values:

    • nowrap (default): single-line which may cause the container to overflow
    • wrap: multi-lines, direction is defined by flex-direction

    flex-wrap: nowrap;

  • flex-flow

    • It contains the combined properties of flex-direction and flex-wrap. We can set two different values at a time with flex-flow.
    flex-flow: row nowrap;

  • align-items

    • We have previously discussed about text-align property. The text-align property was used to align the text, whereas the align-items is used to align items within a container.
    align-items: center;

  • justify content

    • The CSS justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container. We can set different values for this property but basically we use space-between or space-around or space-evenly or center.
    justify-content: space-between;
