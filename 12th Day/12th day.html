Git And Github

  • Git:

    • Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. It is a local system.

  • Github:

    • GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. It is a cloud based system.

Graphical User Interface vs. Command Line Interface

  • Graphical User Interface (GUI):

    • GUI is an interface that allows user to perform action by using different graphical representation like by clicking icon, dragging files/folders etc. It is very easy to use.

  • Command Line Interface (CUI):

    • CUI is an interface that allows user to perform action by typing commands. It is a bit difficult to use than GUI but is faster.

Basic CUI Commands Required For This Course

  • Change Directory

    • To change our current folder (directory) and move to our required directory, we use "cd" command and then the folder name where we want to move.
    Eg: cd folder0

  • Create Folder:

    • To create a folder, we use "mkdir" command which stands for Make Directory. We use mkdir command and then the name of the folder we want to create.
    Eg: mkdir folder1

  • Delete Folder:

    • To remove folder, we use "rmdir" command which stands for Remove Directory. We use rmdir command and the name of the folder we want to delete.
    Eg: rmdir folder1

  • Create file:

    • To create a file we use "echo" then we write the content of our file then we write ">" sign and write our file name and type.
    Eg: echo some_text > file.txt
