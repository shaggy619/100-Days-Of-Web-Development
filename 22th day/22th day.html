22th Day of Web Development

Today's Summary

What We Learn:

In Today's course, we learnt about more concepts of Responsive Web Design. We learnt working with px, %, em and rem value.

Some elements and suitable units:

  • height → rem
  • font-size → rem
  • padding → %
  • border-radius → px

Note: The default value provided by the browser is 16px so we work with other units accordingly.

Media Queries:

We also learnt about the concepts of media queries. The media query code in CSS tells the browser at which device width the media query code should be executed. After reaching a certain width, we can set different value of different elements. 

   @media (min-width or max-width: 1200px) {
        p {
          font-size: 2rem;
          }
        }
