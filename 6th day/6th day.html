Using the Live Server

We can use live server to automatically update the page content when we make changes and save it. Without live server we have to manually reload our page to display new changes but live server does it automatically. To enable live server;
Go to: Extensions < Search "Live Server" < Right click HTML document and click "Open with Live Server".

Understanding Lists

Lists are very useful in HTML page. There are basically two types of Lists. If the list is in specific order then we use order list and for unspecific/random type of list we use unordered list.

Creating Lists

We can creat ordered list and unordered list in different ways. To create ordered list we use ol tag and to create unordered list we use ul tag. We can add items in the ordered/unordered list with li tag.

  • Creating Ordered List:
    • 
        <ol>
          <li>Item1</li>
          <li>Item2</li>
          <li>Item....n</li>
        </ol>
  • Creating Unordered List:
    • 
        <ul>
          <li>Item1</li>
          <li>Item2</li>
          <li>Item....n</li>
        </ul>

Styling Lists

We can add different CSS properties to our list elements. In ordered list by default items are listed with numbers. We can change numbers with uppercase letters, lowercase letters, uppercase roman, lowercase roman or none. In unordered list by default items are listed with bullets. We can change bullets to circle, square or none. We can change the style with "list-style" CSS property. 


            ol {
              list-style-type: none;
              color: #ffffff;
              background-color: #f9004d;
            }
