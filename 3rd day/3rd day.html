Working With Colors

We can add colors to various elements of webpage.
Basically we use following properties:
i) color
ii) background-color
iii) border-color

Formatting Code

We can format our code using tab key or using extension. We can use extension called "Prettier - Code Formatter"
Inorder to add extension go to: File > Preferences > Settings > Search "Format" > Default formatter select "Prettier".

Browser Developer Tool

We can temporarily edit the stuffs in any of the webpage. For this: Right click on any page > Click "Inspect" and play with the code.
We can edit the code in "Elements" tab and select the elemets with arrow enclosed in rectangle icon.

Adding Links

We can add links to a webpage to send the user to another destination when clicked on it. To add link we use 

            <a href = "Link of Destination"> Text You Want To Display </a>
            Example:
            Click here to go to Google

            Click here to go to Github

Using Global CSS Styles

We can add style to the element by declaring it in <head> </head> tag.


       <head>
       <style>
       p {
           font-family: sans-serif;
           text-align: center;
           color: red;
       }
       </style>
       </head>

HTML Document and Metadata

The head section of HTML document holds all the metadata elements which are used in the HTML code but are not visible to the webpage
Similarly, body tag holds all of the page content which are visible to the webpage.

Comments

We can also use comment in our HTML document to make the code clearer. Comment is not diplayed on the webpage but used to make clear a certain code for yourself and for other developers.

Comment format:

<!-- Comment Here -->
