5th Day Of Web Development

What we learn:

  • Adding an Image
  • Styling the Image
  • Styling the Overall Page Background
  • Adding a second HTML file
  • Organizing source files in folders

Adding an Image

We can add image to our HTML document by using img tag. 

<img src="day5.png" alt="Icon of 5th day">
The address of the image is defined by "src" property and "alt" is used to display the text if image loading is failed in HTML page.

Styling the Image

We can use various properties of CSS to style an image. We can set the width and height of the images with width and height property. We can also use border-radius property to change the borders of an Image. Setting border-radius property to 100% results in changing the shape of the image to circle. You can change the value of border-radius to different percentage and play around. 


      img {
      width: 200px;
      height: 200px;
      border-radius: 100%;
      }

Styling the Overall Page Background

We can also change the color and background color of our whole HTML page using background-color property in CSS file. We can also give value to margin property in CSS file which will add spacing to the top, right, bottom and left side of the Webpage. 


      body {
      background-color: #222831;
      color: #ffffff;
      margin: 70px;
      }

Adding a Second HTML File

We can also add our other HTML Files in our HTML document. We can add using anchor tag same as adding the link which we have previously done in our course. 

<a href="D:\Demo Project\1st day\1st day.html">

Organizing Source Files in Folders

We can create different folders in our HTML document to manage our source file and differentiate each file easily. In our case we have created different folders according to different day which will easily help to find out which course was completed on which day.
