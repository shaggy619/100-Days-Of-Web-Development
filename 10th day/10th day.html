HTML & CSS Basics Summary

HTML Summary

HTML (Hyper Text MarkUp Language) is used to define our page content, structure and meaning. We don't use it for styling purposes, we use CSS instead.

  • HTML uses "element" to describe (annotate) content

    • HTML element typically have opening tag, content and closing tag.

  • You can also have void (empty) elements like images
  • You can also configure elements with attributes

    • There's a long list of available elements but don't worry, you will gain experience over time.

Learn more about all available HTML elements on the MDN HTML element reference

CSS Summary

CSS (Cascading Style Sheet) is used for styling our page content.

  • Styles are assigned via property-value pairs.
  • You can assing "style" via the style attribute.
  • To avoid code duplication, we typically use global style (eg. via the "style" element) instead
  • Alternatively, we can work with external stylesheet file which we "link" to
  • When working with CSS concepts like "inheritance", "specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference
