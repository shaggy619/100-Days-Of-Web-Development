Grid Property

Grid property is similar to that of flex property. Flexbox is preferred in one dimensions whereas Grid is preferred in two dimensions. To create a grid container, we set the value of area's container display property to grid.
display: grid;

  • grid-template-columns:

    • The grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column. We can use either pixels or fraction unit to assign value of grid-template-columns property. But typically when we use grid-template-columns, we use fractions unit for value.
    grid-template-columns: 1fr 1fr;

  • gap:

    • The gap property is used to specify the gap/distance between the elements. The first value specifies the space between rows and second value specifies the space between the columns.
    gap: 100px 200px;

  • nth-type selector:

    • The :nth-of-type(n) selector is used to select any list items according to their number. We have to give the value of the position of the list item which is needed to be selected.
