    • Introduction to Box Model

    The box model is used while designing the HTML page. It is basically a box that wraps all the contents around it. It consists of margins, padding, border and the content.

    Using Box Model

    We can use Box Model to set different margins, paddings and border to the page. We have previously used margin in our previous course so today we are going to work with padding and border. Adding padding is similar to that of margin and to add border we have to give first value as its width, second value of its type and third value of its color. 

    
            a {
                border: 1px solid #ffffff;
            }

    Creating HTML Layouts

    It is not a must need but a better practice to add different section in HTML page. It makes easier to differentiate between the contents for ourselves as the code gets long and for other developers who will see our code too. Basically we use header, main and footer in todays course.

    Core Selector

    We have previously discuss about some of the basic selectore like element selector and id selector. Today we are going to learn about few more selectors which are Group selector and Class selector. To use group selector can select two or more elements and set same style to both of the elements by simply using commas between them. 

    
             h3, p {
                 color: #ffffff;
             }
    This will set the color of h3 and p white.
    To add class selector we can simply assign a class to an HTML element and style it according to our choice. Inorder to style a class we have to give dot(.) sign followed by the class name in CSS file. 
                HTML: <li class="style1"><a href="#">No padding</a></li>
            CSS:
            .style1 {
                list-style-type: none;
                margin-bottom: 25px;
            }

    Combinators

    Combinators are the group of selectors. We have used Descendant combinator and Child combinator. In descendant combinator we select element as: 

    
             li p {
                 color: #ffffff;
             }
    Here all p with li as ancestor are selected.
    In child combinator we select element as: 
    
             h2 > p {
                 color: #ffffff;
             }
    Here only p which are direct children of h2 are selected.

    Block vs. Inline elements

    The elements which form new line and takes the full width of webpage are called block elements. For eg: p, h1, li etc.
    The elements which doesn't form new line and only use spaces needed for element are called inline elements. For eg: a, img, span etc.
