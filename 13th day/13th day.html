Git Local Basic Commands

    When we have Git installed in our machine, we need to have knowlege of basic git commands that help to easily organize our file. All these commands are typed on Command Prompt. Select a direcotry where you want to store your project files and folders and then apply all the commands.

  • Set user email:

    • Inorder to use git you must include your email. To set our custom email address we use command:
    git config --global user.email "name@gmail.com"

  • Set user name:

    • We also have to set our user name. To set our username, we use command:
    git config --global user.name "name"

  • git init

    • After installation of Git in our machine we need to type "git init" command which help to turn all the directories into git repository.
    git init

  • Check status:

    • We can check the status of the file, either the file is tracked or untracked. Tracked files are those files which are added/published to the git and untracked files are those files which are not added/published to the git. To check git status we use command:
    git status

  • Add Git

    • We have to add our file to git so that the git recognizes it. To add a file we should use command:
    To add single file:
    git add filename
    To add all untracked files:
    git add .

  • Git Commit

    • To keep track of what have been change we should commit our file. To commit we use commit -m command and write the message what have been changed in the previous file.
    git commit -m "Description"

  • Git Log

    • We can view our commit history using git log command.
    git log

  • Git Checkout

    • We simply delete untracked changes in the files using command git checkout.
    git checkout .

  • Git Reset

    • We can delete our previous commit in tracked file using command git reset.
    git reset file1.txt

Git and Github Basic Commands

  • Linking git and github

    • To publish our code to github, we must link our git to our github repository. To link our git to our github repository we use the command "git remote add origin" followed by our repository link and add ".git" at the end.
    git remote add origin https://github.com/shaggy619/100-Days-Of-Web-Development.git

  • Publishing our files to Github

    • To publish our files to Github, we need to write command "git push origin master". At the first time of using this command, you will be asked to fill up your github email and password and from the next time you won't need it.
    git push origin master

  • Load Files to Local Computer

    • There may also be some situation where you mistakely delete published files on your computer or you want to load the files on another computer, then you can simply do it using command "git clone" followed by the link of your repository.
    git clone https://github.com/shaggy619/100-Days-Of-Web-Development
