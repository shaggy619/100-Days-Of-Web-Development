Text Formatting Tags

Some Text Formatting tags:

  • b:

    • b tag is used to bold text.

  • strong:

    • strong tag is also used to bold text.

  • i:

    • i tag is used to make text italic.

  • em

    • em tag is used to emphasize text.

  • q:

    • q tag defines a short quotation.

  • blockquote:

    • blockquote tag is used

    to display the long quatations.

  • del:

    • del tag is used to denote that some item has been deleted or removed.

  • ins:

    • ins tag defines a text that has been inserted into a document.

  • font:

    • font tag is used to specify the font face, size and color of text

  • pre:

    • pre tag defines 

    preformatted
    text.

  • code

    • code tag is used to define computer code.

  • tt

    • tt tag is used to define teletype text

  • html entities:

    • HTML entities are used to display reserved characters in HTML. For eg: We use &+lt+; to display less than(<) sign and &+gt+; to display greater than(>) sign.

  • sub:

    • sub tag defines subscript text.

  • sup:

    • sup tag defines superscrit text.
